# Extension Icons

This directory contains the icons for the Chrome extension in three sizes:
- `icon16.png` - 16x16px (toolbar icon)
- `icon48.png` - 48x48px (extension management page)
- `icon128.png` - 128x128px (Chrome Web Store)

## Generating Icons

To generate icons from your logo:

1. Open `extension/generate-icons.html` in your web browser
2. Click the "Download" button under each icon size
3. Save the files to this directory with the correct names:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`

The icons are generated using your NAS logo (from `server/static/images/logo.svg`) on a gradient background matching the app's color scheme.
